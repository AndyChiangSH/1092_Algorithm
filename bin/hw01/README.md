## 解題想法
求陣列中的最大值&最小值

1. 一般版本，for迴圈，時間複雜度為O(2n)。
2. 一起算最大值和最小值，時間複雜度為O(n)。
3. 寫在建構元裡面，時間複雜度為O(1)。

因為這次的作業不難，還用不到傳說中的多執行序加速法，下次試試看。
